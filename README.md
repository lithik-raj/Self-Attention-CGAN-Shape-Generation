# 🧠 Self-Attention Conditional GAN for Shape Generation

## 📌 Project Overview

This project implements a **Self-Attention Conditional GAN (CGAN)** using **PyTorch** to generate geometric shapes (Circle, Square, and Triangle) from class labels. A Self-Attention layer is integrated to improve feature learning and visualize attention during image generation.

---

## ✨ Features

- Conditional GAN for shape generation
- Self-Attention mechanism
- Custom shape dataset
- Attention map visualization
- Model training and inference
- Google Colab implementation



## 📂 Dataset

Custom-generated dataset containing:

- 🔵 Circle
- 🟩 Square
- 🔺 Triangle



## 🛠️ Technologies Used

- Python
- PyTorch
- Torchvision
- NumPy
- Matplotlib
- Pillow
- Google Colab

## 📈 Results

- Generated class-conditioned geometric shapes.
- Applied Self-Attention for improved feature learning.
- Visualized attention maps for generated images

- Extend to text-to-image generation
